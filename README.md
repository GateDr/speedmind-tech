# SpeedMind, LLC Website

Static company website for SpeedMind, LLC.

## Files

- `index.html`: company homepage
- `pages/privacy.html`: privacy policy
- `pages/terms.html`: terms of service
- `assets/`: visual assets and CSS
- `CNAME`: custom domain for GitHub Pages-style static hosting

## Stripe readiness notes

Before submitting the website to Stripe, verify whether you want to add:

- Registered business address, if Stripe requires it to be visible.
- A more specific product description once the SpeedMind mobile game is closer to launch.
- Refund/cancellation policy if payments, subscriptions, in-app purchases, or direct sales are added.
- Support response expectations, if customer support will be handled from this domain.
